Mentoring Campus

Project Purpose

A mentoring system for educational institute where students can enrich their career knowledge by linking them with experienced mentor. The platform gives a friendly environment to interact with mentors so that a better counseling is done through a secure channel.

Tools & Technologies

- Ruby 2.3.1
- Rails Framework 4.3.2
- Postgresql for production
- SQLite for developement & testing
- Heroku for deployment
- Amazon S3 for static asset storage
- Private Pub Gem for Realtime Chat Conversation using Faye Server
- JavaScript, Ajax, JQuery, HTML, Bootstrap

Samples of Project

  • Project Demo URL: www.mentoring-campus.heroku.com
  • Completion Date: May 2016
